# The-W-Train-CS395-Project

Problem Statement: To determine whether network traffic can be classified as normal or malicious using features from the UNSW-NB15 dataset.
Null Hypothesis (H₀): There is no relationship between the network features and whether the traffic is malicious or normal.

Alternative Hypothesis (H₁): There is a relationship between the network features and whether the traffic is malicious or normal.
